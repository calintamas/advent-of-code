# Advent of Code 2025

Solutions for [Advent of Code 2025](https://adventofcode.com/2025) in Rust.

To run the solution for a specific day:

```sh
cargo run <day>
```

Examples:

```sh
cargo run 1  # runs day 1
cargo run --release -- 1  # runs day 1 in release mode
```

Input files are automatically loaded from `./input/day{xx}.txt`.
